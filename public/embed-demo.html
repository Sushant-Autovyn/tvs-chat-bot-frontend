TVS Chatbot — Embed Demo

How to embed

Add a single line to the bottom of your page (just before </body>):

<script src="https://tvs-chat-bot-frontend.vercel.app/embed.js" defer></script>

That's it. The floating Support button will appear in the bottom-right corner. Clicking it opens the chat window; the iframe automatically resizes to fit.

Notes

  • Renders inside a sandboxed iframe — zero CSS or JS conflicts with the host page.
  • Safe to include on multiple pages; the loader guards against double-injection.
  • Works on mobile — the chat window clamps to the viewport size.
